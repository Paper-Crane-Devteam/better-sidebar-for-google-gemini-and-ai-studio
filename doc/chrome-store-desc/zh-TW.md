您的 Google AI Studio 歷史記錄是否雜亂無章？
想找回上週寫的那個完美 Prompt 卻無從下手？

Better Sidebar for Gemini & AI Studio 專為開發者和提示詞工程師打造，旨在大幅提升您的工作效率。我們將原本簡單的列表轉化為一個功能強大的知識庫，讓您的 AI 工作流程井井有條。

🔥 最新更新 (v2.1.0)
• 👥 多帳號配置：支援多個 Profile，分別對應不同的 Gemini 和 AI Studio 帳號。
• 📝 增強提示詞庫：支援選項型變數及提示詞嵌套（引用其他提示詞）。
• 🔍 精確搜尋：新增「在當前會話中搜尋」過濾選項，搜尋更有效率。
• ⚙️ 平台管理器：新增外掛程式彈出視窗，快速開啟/關閉平台並一鍵跳轉。

🚀 核心功能

📂 資料夾管理（拖曳式）
建立多層級資料夾，依專案、客戶或主題對對話進行分類。只需拖曳即可整理，保持工作區整潔。

📥 對話匯出
將您的對話匯出為 Markdown、純文字或 JSON 格式，方便分享或存檔。

🏷️ 智慧標籤系統
為對話新增自訂標籤。無論是「程式碼除錯」、「文案寫作」還是「API 測試」，一鍵過濾，快速定位。

🔍 極速搜尋
告別無盡的捲動。透過標題即時搜尋，瞬間找到您需要的歷史對話。

📊 時間軸檢視
透過「今天」、「昨天」、「上週」的時間軸檢視回顧您的工作軌跡，快速接續之前的思路。

⭐ 收藏與置頂
將最常用的 Prompt 或重要對話置頂，隨時呼叫。

🌗 完美融合
原生級的設計體驗，自動同步 AI Studio 的深色/淺色模式。

💾 資料備份與遷移
支援匯出和匯入資料（SQL 檔案），確保您的整理成果安全，或輕鬆遷移到其他裝置。

🔒 隱私優先承諾

• 完全本地化：我們使用瀏覽器內建的 SQLite 資料庫儲存您的資料夾結構。
• 無雲端同步：您的資料夾、標籤和備註不會傳送到我們的伺服器。
• 無資料訓練：我們絕不會使用您的對話內容來訓練 AI 模型。

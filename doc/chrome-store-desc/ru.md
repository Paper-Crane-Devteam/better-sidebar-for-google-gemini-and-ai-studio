🚀 ОСНОВНЫЕ ВОЗМОЖНОСТИ

📂 Папки с Перетаскиванием
Создавайте вложенные папки для группировки чатов по проектам, клиентам или темам. Просто перетаскивайте беседы, чтобы поддерживать рабочее пространство в порядке.

📝 Библиотека Промптов (NEW)
Создайте свою личную библиотеку многоразовых промптов. Используйте переменные (например, {{topic}}) для создания динамических шаблонов и мгновенной вставки их в чаты.

📥 Экспорт диалогов (NEW)
Экспортируйте диалоги в формате Markdown, обычного текста или JSON для документации или резервного копирования.

🔍 Глубокий Поиск (UPGRADED)
Не полагайтесь на память. Используйте полнотекстовый поиск по всей истории сообщений, чтобы найти именно то, что вы ищете.

🏷️ Умная Система Тегов
Назначайте пользовательские теги своим беседам. Будь то "Программирование", "Креативное Письмо" или "Отладка", мгновенно фильтруйте свои чаты.

📊 Вид Временной Шкалы
Визуализируйте свой рабочий процесс с временной шкалой "Сегодня", "Вчера" и "На Прошлой Неделе". Продолжайте точно с того места, где остановились.

⭐ Избранное и Закрепление
Держите наиболее часто используемые промпты наверху для быстрого доступа.

🌗 Безупречная Интеграция
Разработано, чтобы выглядеть и ощущаться как нативное. Автоматически синхронизируется со светлым и тёмным режимами AI Studio.

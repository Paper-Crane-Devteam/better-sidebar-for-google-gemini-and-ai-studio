🚀 核心功能

📂 侧边栏与整理

📂 文件夹与自定义颜色
创建多级文件夹来分类对话，通过拖拽即可整理。支持自定义文件夹和标签的颜色，一目了然。

🔍 搜索与过滤
全文检索所有历史消息。支持按标题、标签、类型等多条件组合筛选，精准定位。可限定"在当前会话中搜索"。

🏷️ 智能标签
为对话添加自定义标签，直接在对话标题处展现，支持实时过滤。

⭐ 收藏与置顶
将最常用的 Prompt 或重要对话置顶，随时高效调用。

📊 时间轴视图
通过"今天"、"昨天"、"上周"的时间轴视图回顾你的工作轨迹。

🗑️ 批量删除
批量删除对话时同步修改 Google 端数据，彻底告别"幽灵对话"。

📥 对话导出
将对话内容导出为 Markdown、纯文本或 JSON 格式，方便分享或存档。

🎨 UI 定制

Gemini：
• 自定义侧边栏、聊天内容、输入框宽度
• 专注模式 — 隐藏多余元素（Logo、免责声明、升级按钮）
• 智能滚动条 — 快速跳转到长对话中的任意消息
• 图片下载无水印（包括 Pro 账号高清图片）

AI Studio：
• 可配置侧边栏宽度
• 自动隐藏输入框
• 自动隐藏运行设置面板

🎨 主题
3 款独家主题皮肤（支持者皮肤包）。自动跟随深色/浅色模式。

💎 GEM、NOTEBOOK 与 PROMPT

💎 Gem 管理与集成
完整的 Gem 集成体验 — 在侧边栏中浏览、筛选、创建 Gem 并直接发起 Gem 对话。

📓 Notebook 集成
完整的 Gemini Notebook 支持 — 在侧边栏中浏览、整理和管理你的 Notebook。

📝 强大提示词库
构建私人的 Prompts 库。支持选项型变量、提示词嵌套组合，以及系统提示词一键导入。

☁️ Google Drive 自动同步
配置和 Prompt 库自动在后台同步到 Google Drive，数据始终有备份。

⚙️ 更多

🌟 双平台支持
在统一的侧边栏中无缝管理 Gemini 和 Google AI Studio 两个平台的历史对话。

👥 多账号配置
支持多个 Profile 配置，每个 Profile 拥有独立的数据库，完美对齐你的 Google 账号。

💾 数据本地可控
支持将个人数据（SQL 归档）随时进行导出备份，全面掌控。

---

🔒 隐私优先承诺

• 完全本地化：我们使用浏览器内置的 SQLite 数据库存储你的文件夹结构。
• 无云端同步：你的文件夹、标签和备注不会发送到我们的服务器。
• 无数据训练：我们绝不会使用你的对话内容来训练 AI 模型。
